# 🐷 PiggyPilot  

PiggyPilot is a **full-stack personal finance and expense tracking app** that helps users manage their income, expenses, and spending patterns in a simple and intuitive way.

---

## 🚀 Live Demo  
🔗 Frontend: https://your-netlify-link  
🔗 Backend API: https://your-render-link  

---

## ✨ Features  

- ➕ Add income and expenses  
- 🗂️ Categorize transactions  
- 📊 Dashboard with financial summary  
- 👤 Authentication (Login / Signup)  
- 📈 Insights like spending patterns  
- 📱 Responsive UI  

---

## 🧠 Tech Stack  

### Frontend  
- React (Vite)  
- Tailwind CSS  
- Axios  

### Backend  
- Node.js  
- Express.js  
- MongoDB (Atlas)  

---

## 📂 Project Structure  
PiggyPilot/
│-- backend/ # Express server & APIs
│-- frontend/ # React application

## ⚙️ Setup Instructions  

### 1️ Clone the repo  
```bash
git clone https://github.com/your-username/piggy-pilot.git
cd piggy-pilot

### 2  backend setup
```bash
cd backend
npm install
npm run dev

### 3  frontend setup
```bash
cd frontend/piggy-pilot
npm install
npm run dev

🌐 Deployment
Frontend deployed on Netlify
Backend deployed on Render
Database hosted on MongoDB Atlas

💡 Note

Render backend may take a few seconds to start due to cold start.
